# Security Policy

## Scope

EdTech Founder Stack is a collection of markdown files (skills and reference data). It does not process user data, run servers, or store credentials. Security concerns for this project are primarily about **data accuracy** rather than traditional software vulnerabilities.

## Reporting data accuracy issues

If you find information in our reference data that is:

- **Incorrect regulatory guidance** (e.g., wrong FERPA interpretation, outdated COPPA requirements)
- **Misleading market data** (e.g., incorrect company information, wrong funding amounts)
- **Outdated research** (e.g., retracted studies, superseded findings)

Please report it by emailing **philippos.savvides@asuep.org** with:

1. The file path and specific content that is incorrect
2. What the correct information should be
3. A source supporting the correction

We will respond within 5 business days and prioritize corrections to regulatory and compliance data.

## Reporting security vulnerabilities

If you discover a security issue in the setup script or any tooling in this repo, please email **philippos.savvides@asuep.org** rather than opening a public issue.
